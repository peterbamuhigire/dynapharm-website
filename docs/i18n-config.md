## Default Language

**en** (English)

## URL Structure

- Root `/` redirects to `/en/` (default language)
- English pages: `/en/about`, `/en/products`, etc.
- French pages: `/fr/about`, `/fr/products`, etc.

## Language Switcher

Display a language switcher in the header with:
- **EN** — English (default)
- **FR** — Français

## Notes

- This website serves Dynapharm Africa's anglophone and francophone markets
- English is the source language — all content is written in English first, then translated to French
- Kiswahili (sw) is disabled for this project
- French translations should use formal francophone African French (vouvoiement — "vous" throughout)
- French text is typically 20-40% longer than English — all designs must accommodate this expansion
- Photos and visual assets are shared across all languages
- The style-brief.md is language-independent and does not require translation
